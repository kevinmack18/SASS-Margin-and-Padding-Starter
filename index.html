OOCSS Spacing Examples

These examples are based on my $default-space in "_variables.scss" being set to 20px.

Basic examples

The naming convention is using the .margin for top/right/bottom/left to receive margin of your default space and .padding for top/right/bottom/left to receive padding of your default space to the container.

The naming convention for a specific side is: ". + (margin or padding ) + (top or right or bottom or left)"

The naming convention for both left and right is: ". + (margin or padding ) + sides"

The naming convention for both top and bottom is: ". + (margin or padding ) + verticals"

Size variations

The naming convention for top, right, left, bottom to receive 2x or half the default space is: ". + (margin or padding ) + -- + (2x or half)"

The naming convention for top, right, left, bottom to receive alternative "smallest size" is: ". + (margin or padding ) + -- + (smallest)"

The naming convention for a specific side to receive modified size variation is: ". + (margin or padding ) + (top or right or bottom or left) + -- + (2x or half or smallest)"

The naming convention for both left and right to receive modified size variation is: ". + (margin or padding ) + sides + -- + (2x or half or smallest)"

The naming convention for both top and bottom to receive modified size variation is: ". + (margin or padding ) + verticals + -- + (2x or half or smallest)"
